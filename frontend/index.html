⚠
This is a pilot / demo stack — not production-hardened.
Before exposing this stack to real workloads or untrusted networks, address every item below.
  • ○Rotate default MinIO credentials
  • ○Enable TLS on all service ports
  • ○Move object storage to managed S3/GCS/Azure
  • ○Configure persistent backups
  • ○Add monitoring + alerting (Prometheus / Grafana)
  • ○Enable RBAC + audit logs
  • ○Restrict network access (firewall / VPN)
  • ○Store secrets in a vault, not .env files
